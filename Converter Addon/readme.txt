ABOUT EMPLOYER 
This is an Add-on application which was request by Terrahotels, a hotel management company. 
This company has its own unique hotel management software built with COBOL. Due to the team 
being very small they have a long list of features that are to be added to this software they 
needed someone contractual to do this add-on.

ABOUT Add-on
This Add-on will be used to make a market study on their hotel clients. The employeer has raw
data in the format of a text file. The objective of this add-on is to take in this raw data
and output a excel file that is "ready to analyse" for the employeer. 

TECHNOLOGY
This application is written in Java and makes use of the APACHE POI API.
