The purpose of the application was to learn Meteor.js, this is the first application I built with Meteor. 
This task was assigned from the company Dengun where I was interning. This application allows users to input
recipes and stores them in a database. In addition if choosen by the user adds it to a shopping list which 
aggregates all the choosed dishes.

This application uses Javascript, Meteor, Mongo DB, JQuery, SASS, HTML, CSS, Bootstrap.
